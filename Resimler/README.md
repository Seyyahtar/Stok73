# Resimler klasörü

Bu klasör, Stok73 MAUI uygulamasının cihaz görselleri veya stok belgeleri gibi yardımcı varlıklarını barındırmak için oluşturuldu. Stok listesini Excel'den içe aktarma özelliği `Resimler/stok.xlsx` dosyasını arar. Lütfen güncel stok dosyanızı buraya kopyalayın veya var olanı güncel tutun.
